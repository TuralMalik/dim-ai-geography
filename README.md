# Lumio Geography

Multilingual MVP for geography exam preparation in Azerbaijani, Russian, and English.

## Included pages

- Dashboard
- Geography diagnostic test
- Test results
- Weak topics
- Topic review
- Progress

## Local development

```powershell
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deployment

The frontend is ready for deployment on Vercel. See `DEPLOY.md` for the required settings.
